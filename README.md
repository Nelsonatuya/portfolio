# Nelson Atuya - Software Engineer Portfolio

A modern, responsive portfolio website showcasing my professional experience, technical skills, and featured projects.

## Features

- **Responsive Design**: Fully mobile-optimized with Tailwind CSS
- **Smooth Animations**: Fade-in effects and typing animations
- **Project Showcase**: Featured projects with links and tech stacks
- **Contact Form**: Easy way for potential clients/employers to reach out
- **Professional Layout**: Clean, modern design with gradient backgrounds

## Technologies Used

- HTML5
- CSS3 / Tailwind CSS
- JavaScript
- View Transitions API

## Sections

- **Home**: Hero section with profile photo and call-to-action buttons
- **About**: Professional overview and key statistics
- **Skills**: Technical competencies with proficiency levels
- **Experience**: Work history and achievements
- **Projects**: Featured projects with descriptions
- **Contact**: Contact information and inquiry form

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize content with your information
